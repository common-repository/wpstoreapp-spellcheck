== Description ==

WPStore.app 中文语法检查插件可以帮助作者获得类似于微信公众号后台的纠错功能，帮助作者写出更好的文章。

= 特性 =

* 支持古腾堡(Gutenberg)编辑器

= 外部服务 =

这个插件接入 [WPStore.app](https://api.wpstore.app) 提供的中文语法检查特性的外部服务。
